Aeon Moon
==========

Port of the classic Aeon theme for modern Pale Moon.

Original theme
--------------
Aeon by Ed Hume.

Mozilla Public License Version 1.1 (MPL 1.1).

Pale Moon port
--------------
This package adapts the original theme for compatibility with modern Pale Moon.
