Aeon Moon 1.1.2
================

Stable release based on the final 1.1.2 beta cycle.

Highlights
----------
- Windows 10/11: custom titlebar buttons now work correctly when the text menu bar is hidden or autohides.
- Final recolored titlebar buttons for minimize, maximize/restore, and close.
- Toolbar button icon-size switching fixed for History popup, Bookmarks popup, and Downloads.
- Bookmarks popup now correctly returns to the large icon when switching back from small icons.
- Keeps the previously verified Aeon Moon 1.1.1b12 base fixes, including Customize Toolbar widget visibility improvements.

Cleanup
-------
- Built from the last fully verified beta state.
- Removed beta numbering and refreshed release metadata.
- Kept only the titlebar and toolbar-icon changes that were confirmed in testing.
